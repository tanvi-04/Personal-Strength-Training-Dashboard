# 🏋️‍♀️ Gym Performance Analytics Dashboard

## 📊 Project Overview
This project analyzes personal workout data to understand **training consistency, strength progression, and workout intensity** using data analytics and visualization.

Using workout logs collected between **October 2025 and March 2026**, I built an interactive **Tableau dashboard** that transforms daily gym activity into meaningful insights about performance trends and training habits.

The goal of this project was to apply **data analytics techniques to real-life behavioral data** and demonstrate how structured tracking can reveal patterns in fitness performance.

# 📁 Dataset

The dataset contains daily workout logs including exercise details, weights lifted, repetitions, and calculated training volume.

### Data Fields

| Column | Description |
|------|------|
| 📅 Date | Workout date |
| 🏋️ Workout Type | Training split (Upper / Lower) |
| 💪 Exercise | Exercise performed |
| 🔍 Muscle Group | Target muscle group |
| 🔁 Sets | Number of sets |
| 🔢 Reps | Repetitions per set |
| ⚖️ Weight_lbs | Weight used in pounds |
| 📦 Volume | Training load (Weight × Reps) |

---

# 📈 Key Performance Indicators (KPIs)

The dashboard tracks several performance metrics:

### 🔄 Workout Consistency Rate
Percentage of days the gym was attended during the tracked period.

### 📊 Strength Improvement
Percentage increase in average weight lifted across the training period.

### 📈 Training Load Growth
Increase in total weekly training volume over time.

### 🔥 Current Workout Streak
Number of consecutive days with logged workouts.

### 🏆 Personal Record Lift
Maximum weight lifted during the tracking period.

---

# 📊 Dashboard Visualizations

The Tableau dashboard includes the following visual analytics:

### 📉 Strength Progression Trend
Monthly trend of average weight lifted to visualize strength improvement over time.

### 📅 Weekly Training Volume
Time-series analysis of total workout volume to track workload changes.

### 💪 Training Volume by Muscle Group
Comparison of workload distribution across different muscle groups.

### 🎯 Intensity vs Volume Analysis
Scatter plot analyzing relationships between workout intensity and total training load.

# 🎯 Project Motivation

Fitness tracking apps typically focus on logging workouts but rarely provide deeper analytical insights.

This project demonstrates how **data analytics can be applied to personal performance tracking**, turning everyday data into actionable insights about consistency, workload, and improvement trends.
